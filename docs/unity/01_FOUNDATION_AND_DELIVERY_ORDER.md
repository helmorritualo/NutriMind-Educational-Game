## Asset and UI Reuse Rule

Before creating UI:

1. inspect existing scene designs;
2. inspect existing Canvas prefabs;
3. inspect UI Toolkit assets;
4. inspect image references;
5. reuse what exists;
6. create only missing screens/components.

Do not rebuild finished UI from scratch unless it is broken or not reusable.
