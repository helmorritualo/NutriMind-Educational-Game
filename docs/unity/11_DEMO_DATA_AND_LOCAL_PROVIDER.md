# LocalDemoJson — Quiz-First Laravel Contract

## Purpose

Simulate the Laravel REST contract for Unity quiz testing.

## Simulate

- login;
- logout;
- refresh/bootstrap;
- profile;
- settings;
- subjects;
- terms;
- quizzes;
- quiz details;
- attempts;
- results;
- sync status;
- reset.

## Do Not Simulate

- mission progress;
- rewards shop;
- inventory;
- world restoration.

## Production Guard

Production builds must reject LocalDemoJson.
