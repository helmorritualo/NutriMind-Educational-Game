# Handoff and Stop
Do not work simultaneously on the same files as OpenCode or another Unity agent. Report the selected unit, changed files/assets, scene/prefab/component changes, reused assets, variants, generated assets, Canvas migrations, placeholders, shared systems, compilation/test results, known gaps, pending server/OpenCode work, and proposed next unit. Stop and wait for approval.
