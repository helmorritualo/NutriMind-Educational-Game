# One-Unit Scene Workflow
Confirm the selected unit and prerequisites. Inspect existing scenes, prefabs, registries, references, and assets. Re-scan third-party and provided assets. Implement only the selected unit using shared navigation, scene registry, player/camera baseline, world HUD, station framework, attempt coordinator, progress/reward stores, and mission/NPC/hint/feedback components. Save, compile, validate references, run focused Play Mode tests, and do not modify the next scene.
